# 🚀 Smart FinTech Fraud Detection System

## 📌 Overview
This project is a full-stack FinTech application designed to detect fraudulent transactions in real-time. It integrates a user-friendly frontend interface with a powerful backend powered by machine learning and FastAPI.

The system analyzes transaction patterns and flags suspicious activities, helping improve financial security and reduce fraud risks.

---

## 🎯 Key Features
- 🔍 Real-time fraud detection
- 💳 Secure payment simulation interface
- ⚡ FastAPI-based backend for high performance
- 🧠 Machine Learning model for prediction
- 🗄️ MongoDB integration for data storage
- 📊 Risk analysis and decision output
